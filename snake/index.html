Use arrow keys or WASD to move. Press SPACE to pause. Press ESC to exit to the main menu.
		
Be sure to try out the level editor.
		
Editor keys:
	1 -> 6, 7 -> = : select block type
	Primary	                  Secondary
	1: empty block            7: empty block       L: toggle link mode
	2: wall                   8: wall              M: set link mode: pair, chain, star
	3: bomb                   9: bomb              R: toggle run
	4: food                   0: food              
	5: teleporter             -: teleporter        Q: set cursor type
	6: inverter               =: inverter          [ ] or mouse scroll: set cursor size
